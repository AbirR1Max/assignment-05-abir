What are the different ways to select an element in the DOM?

In JavaScript, there are several ways to select elements in the DOM:
getElementById(): Selects an element with a specific id.
getElementsByClassName(): Selects all elements with a specific class name.
getElementsByTagName(): Selects all elements with a specific tag name.
querySelector(): Selects the first element that matches a given CSS selector.
querySelectorAll(): Selects all elements that match a given CSS selector.
getAttribute(): If you need to select an element by a specific attribute.

What is the difference between innerHTML, innerText, and textContent?
innerHTML: Gets or sets the HTML content inside an element (including HTML tags).
Allows HTML markup.
Can be a security risk if not properly sanitized.
innerText: Gets or sets the visible text content of an element, excluding HTML tags.
The result may be influenced by CSS.
It represents the text content as it's displayed to the user.
textContent: Gets or sets the text content of an element, including hidden text.
Unlike innerText, it does not consider CSS styling.
Faster than innerText because it doesn't trigger reflow or layout updates.
What is event bubbling in the DOM?
Event bubbling is the process by which an event, after being triggered on an element, propagates (or "bubbles") up to its parent elements in the DOM tree. This means the event is first captured by the target element and then passed up to its ancestors, triggering the same event on each parent element.
Example: If you click a button inside a div, the click event bubbles from the button up to the div, then the body, and finally to the document. To prevent event bubbling, you can use event.stopPropagation().
How do you create, add, and remove elements using JavaScript?
Creating an element: Use document.createElement() to create an element. Adding an element: Use appendChild() or insertBefore() to add an element to the DOM. Removing an element: Use removeChild() or remove() to remove an element.
